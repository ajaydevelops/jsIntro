Variable

02 Apr 2022

Three types of variable in javaScript.

  • var
  • let
  • const

Var : Var keyword ka use variable ko declear karne ke liye kya jahta h. Var function scope hota h. Var mutable hota h.

Example:


        let a = 10;
        console.log(a);
