javaScript:

What is javaScript:

javaScript is a programming and synchronous language based on behavior.

Need of javaScript:

To create web pages.

Use of javaScript:

  • To create mobile applications.
  • To create desktop applications.

File extension of javaScript:

.js

There are two ways to add javaScript inside HTML file:

  • Internal.
  • External.

Adding internal javaScript inside HTML file:

To add internal javaScript inside the HTML file, script tag is used inside the head tag. Internal javaScript can only be added inside a HTML file. Can't reuse internal javaScript.

Example:

                            
<html>
    <head>
        <title>Internal javaScript</title>
        <script>
            let p = document.getElementsByTagName("p");
            p.style.color = "green";
        </script>
    </head>
    <body>
        <p>Internal javaScript can only be added inside a HTML file.</p>
    </body>  
</html>

Adding external javaScript inside HTML file:

To add external javaScript inside the HTML file, script tag is used inside the body tag. External javaScript can be added inside multiple HTML files. Can reuse external javaScript.

Example:

                            
<html>
    <head>
        <title>External javaScript</title>
    </head>
    <body>
        <p>External javaScript can be added inside multiple HTML files.</p>
        <script src="javaScript.js"></script>
    </body>
</html>
