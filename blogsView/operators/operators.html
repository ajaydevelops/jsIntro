Arithmetic Operators:

Arithmetic Operators:   Arithmetic operators ka use mathematical operation ke lie kya jaata hai.

Operator Operator Name
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus
++i Pre Increment
i++ Post Increment
--i Pre Decrement
i-- Post Decrement

Example:

                
let a = 10;
let b = 2;
let c;

c = a + b;
console.log(c);

c = a - b;
console.log(c);

c = a * b;
console.log(c);

c = a / b;
console.log(c);

c = a % b;
console.log(c);

++c;
console.log(c);

c++;
console.log(c);

--c;
console.log(c);

c--;
console.log(c);
