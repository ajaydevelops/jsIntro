Operators:

What is operators:

Operators are symbol that performs operations.

Need of operators:

To performs operations.

Type of operator:

  • Assignment Operators
  • Arithmetic Operators
  • Comparison Operators
  • Logical or Boolean Operators
  • String Operators

Assignment Operators:

Assignment Operators:  Assignment Operators performs assignment operations between two operands.

Operator Operator Name Example Description
= Equal x = y x = y
+= Addition equal x += y x = x + y
-= Substraction equal x -= y x = x - y
*= Multiplication equal x *= y x = x * y
/= Division equal x /= y x = x / y
%= Modulus equal x %= y x = x % y

Example:

                            
let x = 20;
let y = 10;
console.log(x, y);

x += y;
console.log(x);

x -= y;
console.log(x);

x *= y;
console.log(x);

x /= y;
console.log(x);

x %= y;
console.log(x);

Output:

                            
// 20 10
// 30
// 20
// 200
// 20
// 0

Arithmetic Operators:

Arithmetic Operators:  Arithmetic Operators performs mathematical operations between two operands.

Operator Operator Name Example
+ Addition a + b
- Substraction a - b
* Multiplication a * b
/ Division a / b
% Modulus a % b
++ Pre-Increment ++a
Post-Increment b++
-- Pre-Decrement --a
Post-Decrement b--

Example:

                            
let a = 10;
let b = 5;
let addition = a + b;
console.log(addition);

let substraction = a - b;
console.log(substraction);

let multiplication = a * b;
console.log(multiplication);

let division = a / b;
console.log(division);

let modulus = a % b;
console.log(modulus);

++a;
let preIncrement = a;
console.log(preIncrement);

b++;
let postIncrement = b;
console.log(postIncrement);

--a;
let preDecrement = a;
console.log(preDecrement);

b--;
let postDecrement = b;
console.log(postDecrement);

Output:

                            
// 15
// 5
// 50
// 2
// 0
// 11
// 6
// 10
// 5

Comparison Operators:

Comparison Operators:  Comparison Operators performs comparison operations between two operands and returns a boolean value. If the Condition is true then the Comparison Operators returns true and if the condition is false then the Comparison Operators returns false.

Operator Operator Name
== Equal to
=== Equal to and Same type
!= Not equal to
!== Not equal to or Not same type
> Greater than
>= Greater than or Equal to
< Less than
<= Less than or Equal to
Equal to Operator(==):

Equal to Operator:  If the both operands of Equal to Operator are equal then the Equal to Operator returns true otherwise returns false.

Example:

                                
let x = 10;
let y = 10;
let z = x == y;
console.log(z);

y = 20;
z = x == y;
console.log(z);

Output:

                                
// true
// false
Equal to and Same type Operator(===):

Equal to and Same type Operator:  If the both operands of Equal to and Same type Operator are equal and have the same type, then the Equal to and Same type Operator returns true otherwise returns false.

Example:

                                
let x = 10;
let y = 10;

console.log(typeof x);
console.log(typeof y);

let z = x === y;
console.log(z);

x = "10";
console.log(typeof x);

z = x === y;
console.log(z);

Output:

                                
// number
// number
// true
// string
// false
Not equal to Operator(!=):

Not equal to Operator:  If the both operands of Not equal to Operator are not equal then the Not equal to Operator returns true otherwise returns false.

Example:

                                
let x = 10;
let y = 20;
let z = x != y;
console.log(z);

y = 10;
z = x != y;
console.log(z);

Output:

                                 
// true
// false
Not equal to or Not same type Operator(!==):
Condition A:

If the both operands of Not equal to or Not same type Operator are not equal and their types are also not the same, then the Not equal to or Not same type Operator returns true otherwise returns false.

Example:

                                    
let x = 10;
let y = "20";

console.log(typeof x);
console.log(typeof y);

let z = x !== y;
console.log(z);

x = "20"
console.log(typeof x);

z = x !== y;
console.log(z);

Output:

                                    
// number
// string
// true
// string
// false
Condition B:

If the both operands of Not equal to or Not same type Operator are equal but their types are not the same, then the Not equal to or Not same type Operator returns true otherwise returns false.

Example:

                                    
let x = 10;
let y = "10";

console.log(typeof x);
console.log(typeof y);

let z = x !== y;
console.log(z);

x = "10";
console.log(typeof x);

console.log(z);

Output:

                                    
// number
// string
// true
// string
// false
Condition C:

If the both operands of Not equal to or Not same type Operator are not equal but their types are the same, then the Not equal to or Not same type Operator returns true otherwise returns false.

Example:

Logical or Boolean Operators:

Logical or Boolean Operators:  Logical or Boolean Operators performs logical or boolean operations between two operands and returns a boolean value. If the Condition is true then the Logical or Boolean Operators returns true and if the condition is false then the Logical or Boolean Operators returns false.

Operator Operator Name
&& AND
|| OR
! NOT
AND Operator(&&):

AND Operator:  If the both operands of AND Operator are true then the And Operator returns true otherwise returns false.

Example:

                                
let a = 25;
let b = 15;
let c = 25;
let d = (a > b) && (a == c);
console.log(d);

a = 24;
d = (a > b) && (a == c);
console.log(d);

Output:

                                
// true
// false
OR Operator(||):

OR Operator:  If the both operands of OR Operator are true or one of the both operands be true then the OR Operator returns true otherwise returns false.

Example:

                                
let a = 10;
let b = 20;
let c = 30;
let d = (a <= b) || (b != c);
console.log(d);

c = 20;
d = (a <= b) || (b != c);
console.log(d);

a = 25;
d = (a <= b) || (b != c);
console.log(d);

Output:

                                
// true
// true
// false
NOT Operator(!):

NOT Operator:  NOT Operator has single operand if operand is true then the NOT Operator returns false and if operand is false then the NOT Operator returns true.

Example:

                                
let a = 10;
let b = 10;
let c = (a === b);
console.log(c);

c = (! a === b);
console.log(c);

Output:

                                
// true
// false

String Operators(+):

String Operators:  String Operators performs string operations between two operands, one of which operand is string value and the other operand is string value or other data types value. String Operators concatenate the operands values inside the single string.

Example:

                            
let name = "amit";
let age = 26;
let isYoung = true;
let stringOperators = name + age + isYoung;
console.log(stringOperators);

Output:

                            
// amit26true
